Internationalisation
====================

RBAM fully supports i18n
- though at the time of writing the only supported language is English (and it is English, not American);
if anyone would like to provide translations to another language or languages, you will find all the messages in
`./resources/messages/en-gb/rbam.php` relative to RBAM's root directory;
fork the repository, do the work, then create a pull request. Same goes for this documentation.

.. note::

    Internationalisation must be configured for the application.

In addition to RBAM itself, RBAM can also translate RBAC Permission, Role, and Rule descriptions.

.. note::

    Names are not translated.

RBAM uses three categories for translation:

* For internal messages the `rbam` category is used
* For RBAC item names and descriptions the `rbac-item` category is used
* For RBAC rule descriptions the `rbac-rule` category is used

The `rbam` category translations are in `./resources/messages` relative to RBAM's root directory.

The other two categories are in `./rbac/translations` relative to RBAM's root directory.
