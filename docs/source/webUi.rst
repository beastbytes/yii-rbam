RBAM Web UI
-----------

RBAM provides a powerful yet easy to use web based UI to manage RBAC Permissions, Roles, snd Rules, RBAC hierarchy,
and Role Assignments to Users.

RBAM fully supports i18n in the UI to localise RBAC Permission, Role, and Rule descriptions.
