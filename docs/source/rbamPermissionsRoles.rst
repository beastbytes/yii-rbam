RBAM Permissions and Roles
==========================

This section lists the RBAC Permissions and Roles in RBAM.

Permissions
-----------

rbam.clear
    Clear RBAC Assignments, Permissions, and Roles

    Granted by rbam.admin

rbam.index
    View RBAM dashboard

    Granted by rbam.admin, rbam.item.manager, rbam.rule.manager, rbam.user.manager

rbam.item.create
    Create Permissions and Roles

    Granted by rbam.item.manager

rbam.item.remove
    Remove Permissions and Roles

    Granted by rbam.item.manager

rbam.item.update
    Update Permissions and Roles

    Granted by rbam.item.manager

rbam.item.view
    View Permissions and Roles

    Granted by rbam.item.manager

rbam.rule.create
    Create Rules

    Granted by rbam.rule.manager

rbam.rule.delete
    Delete Rules

    Granted by rbam.rule.manager

rbam.rule.update
    Update Rules

    Granted by rbam.rule.manager

rbam.rule.view
    View Rules

    Granted by rbam.rule.manager

rbam.user.update
    Update Users

    Granted by rbam.user.manager

rbam.user.view
    View Users

    Granted by rbam.user.manager

Roles
-----

rbam.admin
    Perform all RBAM functions

rbam.item.manager
    Create, update, delete, and view RBAC Roles and Permissions

rbam.rule.manager
    Create, update, delete, and view RBAC Rules

rbam.user.manager
    View users, and assign and revoke RBAC Roles
